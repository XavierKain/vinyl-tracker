# Vinyl Collection Tracker PWA

## Goal
A React PWA to track vinyl record collections — completely offline, no Discogs account needed. Sell for $9 one-time on Gumroad.

## Market Context
- r/vinyl has 1.2M members, many frustrated with Discogs app after their redesign
- Multiple posts specifically asking "best app for tracking LPs (no discogs!)"
- People use Google Sheets as fallback — clearly no good alternative

## Core Features (MVP — 1 night)
1. **Add record** — manual OR scan barcode via device camera
   - Barcode → auto-lookup via MusicBrainz API (free, no auth): `https://musicbrainz.org/ws/2/release/?query=barcode:XXXXX&fmt=json`
   - Fields: Artist, Album, Year, Format (LP/EP/45), Condition (M/NM/VG+/VG/G+/G), Color/Variant, Purchase price, Purchase date, Notes
2. **Collection grid** — searchable, filterable by artist/year/condition
3. **Record detail view** — all fields + edit + delete
4. **Collection stats** — total records, total value paid, breakdown by format/condition
5. **Import/Export CSV**
6. **PWA** — installable on mobile (manifest + service worker), offline first
   All data in IndexedDB — no backend, no account

## MusicBrainz API
- Barcode: `https://musicbrainz.org/ws/2/release/?query=barcode:0724356956828&fmt=json&limit=5`
- No auth, rate limit 1 req/sec

## Tech Stack
- React 18 + TypeScript + Vite + TailwindCSS
- Dexie.js (IndexedDB wrapper)
- @zxing/browser (barcode via camera)
- vite-plugin-pwa
- Dark mode default
